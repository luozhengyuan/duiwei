# DESIGN.md - 对味 (DuiWei)

## 1. 品牌与视觉方向

### 产品定位
「对味」是一款 AI 驱动的线上社交聊天训练工具，帮助用户练习开场白、延续话题、情绪识别等核心社交技能。

### 视觉关键词
**约会App风格** — 简洁、现代、年轻化、有温度。像 Tinder/Bumble 的干净感，但更有学习产品的专业性。

### 色彩系统
| 角色 | 色值 | 用途 |
|------|------|------|
| 主色 | `#FF6B6B` | 珊瑚红，代表热情与社交能量 |
| 辅助色 | `#4ECDC4` | 青绿色，代表成长与积极 |
| 强调色 | `#FFE66D` | 暖黄色，用于高亮和评分 |
| 背景色 | `#FAFAFA` | 近白色，干净舒适 |
| 深色文字 | `#1A1A2E` | 深海军蓝，沉稳专业 |
| 浅色文字 | `#6B7280` | 灰色，次要信息 |
| 卡片背景 | `#FFFFFF` | 纯白卡片 |
| 成功色 | `#10B981` | 绿色，加分项 |
| 警告色 | `#F59E0B` | 橙色，踩雷项 |

### 字体系统
- **标题**: Inter Bold (700), 回退 sans-serif
- **正文**: Inter Regular (400)
- **数字/评分**: Inter Semibold (600)

### 动效原则
- 页面切换：淡入淡出 200ms ease-out
- 卡片悬停：轻微上浮 translateY(-2px)，配合阴影加深
- 消息气泡：弹性出现 scale(0.95→1) + opacity(0→1)，150ms
- 按钮点击：轻微缩小 scale(0.98)，100ms
- 打字指示器：三个圆点依次跳动，间隔 300ms

## 2. 布局与结构

### 页面层级
```
首页 (/) → 场景选择 (/scenes) → 角色选择 (/scenes/[id]) → 对话页 (/chat/[sessionId]) → 复盘页 (/review/[sessionId])
```

### 首页设计
- 顶部：品牌 Logo + 标语 "聊出对的味道"
- 中部：快速开始按钮 + 今日推荐场景
- 底部：底部导航栏（首页/练习记录/我的）

### 对话页设计
- 顶部：返回按钮 + 对方头像 + 对方状态（打字中/在线）
- 中部：消息气泡列表，滚动区域
- 底部：输入框 + 发送按钮 + 候选回复抽屉

### 复盘页设计
- 顶部：总评分（圆形进度环）
- 中部：关键节点卡片（加分/踩雷）+ 改进建议
- 底部：重新练习 + 返回首页

---

## 3. 组件规范

### 场景卡片
- 尺寸：全宽，高度 120px
- 内容：场景图标 + 场景名称 + 场景描述 + 难度标签
- 状态：默认、悬停（阴影加深）、已解锁/未解锁

### 角色卡片
- 尺寸：全宽，高度 80px
- 内容：头像（圆形，60px）+ 昵称 + 性格标签 + 在线状态
- 悬停：左侧出现箭头指示

### 消息气泡
- 用户消息：右对齐，主色背景，白色文字
- AI 消息：左对齐，白色背景，深色文字，带小三角
- 时间戳：消息下方居中，浅色小字

### 候选回复选项
- 出现在输入框上方
- 3-4 个选项横向排列或垂直列表
- 点击选中后复制到输入框

### 复盘卡片
- 加分项：左边绿色条 + 成功图标
- 踩雷项：左边橙色条 + 警告图标
- 内容：原消息 + 分析 + 改进建议

---

## 4. 交互动效

### 消息发送
1. 用户点击发送 → 消息立即出现在列表（乐观更新）
2. 显示"发送中"状态
3. AI 回复到达 → 打字指示器动画
4. AI 回复完成 → 打字指示器消失，气泡出现

### 复盘展开
- 点击关键节点卡片 → 展开详细内容
- 展开动画：高度从 0 到 auto，200ms ease-out

---

## 5. 响应式策略

- **移动端优先**：主要设计考虑 375px-428px 宽度
- **平板/桌面**：最大宽度 480px 居中，保持移动端体验
- **底部导航**：移动端固定在底部，桌面端可隐藏

---

## 6. 设计禁忌

- ❌ 不要使用深色背景（保持轻盈感）
- ❌ 不要使用过于复杂的渐变
- ❌ 不要使用 emoji 作为主要图标（用 Lucide Icons）
- ❌ 不要过度动画（保持专业感）
